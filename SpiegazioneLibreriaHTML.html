Introduzione alla LIBRERIA

La LIBRERIA è un archivio di dati che ti consente di inserire libri, eliminarli, modificarli e cercarli per alcuni criteri.

Installazione della libreria

Essendo un programma scritto in java, assicurati di avere Java installato sulla tua macchina. Successivamente, segui questi passaggi:

  1. Scarica il file JAR dalla repository GitHub che trovi alla fine della pagina.
  2. Apri il terminale e naviga nella cartella dove hai scaricato il file.
  3. Esegui il comando: java -jar Libreria.jar.

Manuale dell'utente

Ecco qualche aiuto per iniziare ad archiviare dei libri_

Aggiungere un libro

Per aggiungere un libro alla libreria, clicca sul pulsante "Aggiungi libro", inserisci i dettagli richiesti (titolo, autore, genere), e conferma.

Ricerca per titolo

Puoi cercare un libro per titolo cliccando sul pulsante "Cerca". Inserisci il titolo del libro e visualizza i risultati.

\

Come è stato creato la Libreria?

La libreriaJAVA, come nel suo nome, è stata sviluppata utilizzando Java tramite l'uso della libreria Swing per la creazione dell'interfaccia grafica e dei file di testo per la memorizzazione dei dati.

Gestione dei Dati

I dati dei libri vengono memorizzati in un file di testo, dove ogni libro è rappresentato da un blocco di informazioni che contiene il titolo, autore, e altre caratteristiche.
